Como obter o Client ID:
1. Acesse console.cloud.google.com
2. Crie um projeto → APIs & Serviços → Credenciais
3. Crie ID do cliente OAuth 2.0 (Aplicativo Web)
4. Adicione a URL desta página como URI de redirecionamento
